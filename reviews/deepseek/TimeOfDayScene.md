Level 25 — Time of Day
The day/night enemy sleep mechanic is beautifully simple, but first-encounter fairness is compromised because the CYCLE TIME button is small and bottom-right—players may not notice it while focused on the jump. The completability math is sound: the guard spike patrols the only safe landing of the forced P1->P2 jump (gap 130pt, rise 13pt), so in DAY mode the spike is lethal but timing-able, and NIGHT mode disables the hazard entirely. The hint quality is excellent: hintText() explicitly names the CYCLE TIME button, and the instruction panel's "SOMETHING GUARDS THAT LEDGE" line introduces the threat without spoiling the solution. A specific code bug: overrideMode is set in cycleTimeOverride(), but applyTimeMode() reads overrideMode directly—if the player manually cycles to NIGHT, then the real clock hits NIGHT, the override mode persists and prevents the real time from taking effect. One concrete improvement: reset overrideMode to nil when the player reaches the exit platform, so subsequent attempts on the same level start with the real clock time rather than the last override state.
