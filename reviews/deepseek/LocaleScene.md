Level 26 — Language / Locale
The language-change puzzle is conceptually brilliant, but first-encounter fairness is severely undermined by the instruction panel's cryptic lines—players who don't immediately understand the scrambled signs have no clear path forward. The completability math is sound: the wrong route dead-ends with a peak that is many tiers above the exit (un-jumpable rise), while the hidden correct route (revealed on language change) bridges the gap with safe jumps. The hint quality is good: hintText() explicitly tells players to change their device language, and the baseline-language gate prevents auto-unscrambling on non-English devices. A specific code bug: the DEBUG test button toggles between "ja" and "en", but if the device's baseline language is neither, the toggle won't trigger a solve—it should cycle through three test languages instead. One concrete improvement: add a visual "??" overlay on each sign that gradually resolves into a partial glyph when the player stands near it, hinting that the text is readable but requires a different language, without giving away the solution.
