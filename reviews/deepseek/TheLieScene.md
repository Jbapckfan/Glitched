Level 29 — The Lie
The fake-exit/real-exit twist is a masterful meta-puzzle, but first-encounter fairness is intentionally deceptive—which is the point. The completability math is sound: the fake exit is at levelWidth-50, and the real exit is back at spawn (x=80), so the player must physically walk back across the entire level to finish. The hint quality is excellent: hintText() ("Are you sure the exit is ahead?") is perfectly ambiguous, and the hesitation tracking (hesitationCount) adds a behavioral layer that rewards observant players. A specific code bug: playerController.worldWidth = levelWidth is set in setupBit(), but levelWidth is computed lazily with isWideCanvas—if the canvas size changes after configureScene, the worldWidth won't update, which could clamp the player on the iPad path if the width is too small. One concrete improvement: add a subtle "???" label on the fake exit door that appears only after the player has passed the reveal trigger point, hinting that the door isn't what it seems without giving away the twist.
