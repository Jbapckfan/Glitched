Level 17 — Airplane Mode
The death-plane trap that makes flying platforms unreachable when Airplane Mode is OFF is clever, but first-encounter fairness is weak because the instruction panel's atmospheric lines ("CUT THE WORLD LOOSE") don't clearly signal that toggling a system feature is the solution—players may try jumping or looking for an in-world button. The completability math works: landed platforms sit inside the death zone at groundY-220, so toggling OFF while standing on one would kill the player, which the flyingPlatformSupportingBit() protection correctly prevents. The hint quality is solid: the explicit Airplane Mode instruction lives in hintText() and escalates through notePlayerStruggle() on death. A specific rough edge: the turbulence animation (flyingPlatforms wobble when active) adds 1.5-2pt of position jitter, which can cause a grounded Bit to lose contact briefly and trigger a false airborne state. One concrete improvement: make the turbulence purely visual by moving a child node inside each flying platform, leaving the platform's physics body position stable, so Bit's grounded state never flickers during the wobble.
