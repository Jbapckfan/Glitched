Level 24 — Storage Space
The two-step purge (reach terminal -> arm -> clear storage) is well-designed, but first-encounter fairness suffers because the instruction panel's "JUMP TO THE PURGE TERMINAL FIRST" is the only cue—players may tap "CAN'T DO THIS?" immediately and wonder why nothing happens. The completability math is sound: the junk wall is floor-to-ceiling (topSafeY-110) and un-jumpable, and the raised terminal ledge (60pt rise from start) requires a deliberate jump, so the player must engage with the level geometry. The hint quality is excellent: hintText() changes based on terminal state ("Jump to the terminal" vs "Free up storage"), and notePlayerStruggle() on death correctly escalates the reveal. A specific code bug: StorageSpaceManager.shared.removeCacheFile() is called in willMove, which can delete the cache prematurely if the player exits the level—this should only run on a successful purge or when the scene is definitively torn down. One concrete improvement: add a small "ARMED" indicator light on the terminal that turns green when the player stands near it, giving immediate feedback that the purge is now ready to trigger.
