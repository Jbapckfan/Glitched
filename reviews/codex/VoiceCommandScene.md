Level 22 — Voice/speech recognition

The t=0 copy says to speak to the phone but not which words, so the first encounter may be confusing until the player sees the BRIDGE/OPEN/FLY gates or earns the hint. The geometry supports the intended order: the voice chasm is unjumpable, the door blocks progress, and the FLY rise is too high to clear normally. Deaths call `notePlayerStruggle()`, and each successful command calls `notePlayerProgress()`, which is strong hint plumbing. The no-mic fallback is thoughtful and only appears when the mic path appears to be failing, but the buttons also reveal all command words and can flatten the puzzle. A rough edge is that the spoken command vocabulary includes aliases like JUMP for FLY, which can accidentally trigger the final mechanic if the player is experimenting aloud. One concrete improvement: add in-world command labels at each gate after one failed/unknown utterance, rather than exposing the full fallback command row immediately.
