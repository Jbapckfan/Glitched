Level 33 — App Store review prompt / meta finale

The finale is fair because the flat level looks simple, then the two joke gates and optional validation button reveal the real meta mechanic. There are no meaningful jumps or gaps; the iPad path deliberately preserves the flat layout and only lifts the whole band for framing. The t=0 "JUST GET TO THE EXIT" copy avoids spoiling the review ask, and the terminal eventually says the validation is optional before the auto-unlock. The hint text is too vague, and `handleDeath()` does not call `notePlayerStruggle()`, so repeated failures do not escalate toward useful help. The largest design mismatch is that no real App Store review prompt is requested; `VALIDATE ME` is a fictional in-game button, with `.appReviewReturned` handled only as a possible external event. One concrete improvement: either wire a compliant `SKStoreReviewController` path with the same 10-second optional unlock, or rename the mechanic/copy to make clear this is a simulated validation prompt rather than an App Store review.
