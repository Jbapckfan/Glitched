Level 25 — Real device clock

The clock mechanic is understandable from "THE WORLD CHANGES WITH THE CLOCK", and the guard sleeping at night is a fair rule once observed. The phone route's forced landing gap is right at the 130 open-gap budget, and the iPad route uses single-tier steps, so reach is acceptable but the phone teach jump has little spare margin. Deaths call `notePlayerStruggle()`, and cycling time records progress, so the hint system will not nag after the player starts using the mechanic. The rough design edge is the always-visible `CYCLE TIME` button: it makes the real-clock level trivially solvable without changing the device time. The code also uses a force unwrap on `overrideMode!` immediately after assignment, which is safe in context but unnecessary. One concrete improvement: make `CYCLE TIME` a fallback that appears after struggle or accessibility mode, while the first solution path reads from the actual clock.
