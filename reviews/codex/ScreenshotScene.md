Level 7 — Taking a screenshot (Side Button + Volume Up)

The opening line "SOME MOMENTS REFUSE TO MOVE" is a good non-spoiler clue, and the hint later names the screenshot gesture directly. The 220-point chasm is intentionally unjumpable, but the shortest degraded freeze is only about 1.0 second plus grace, so late retries can become tighter than the mechanic needs. The iPad course keeps normal approach jumps safe and preserves the same screenshot-freeze finale. Deaths call `notePlayerStruggle()` and screenshot count resets on death, which keeps the first retry from being punished too hard. The rough edge is the combination of freeze degradation and a two-second screenshot cooldown, which can make a failed learning attempt feel like system latency rather than puzzle feedback. One concrete improvement: keep the minimum freeze at 1.5 seconds and add clearer start/end markers for the frozen crossing.
