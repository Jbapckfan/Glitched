Level 21 — Device name reading (Settings > General > About > Name)

The opening "I KNOW WHO YOU ARE" plus "FIND THE DOOR WITH YOUR NAME" makes the mechanic very readable, though it largely spoils the device-name premise. Platforming looks safe, with the phone route using small gaps/rises and the iPad route generating one-tier steps within the 85-rise limit. Deaths call `notePlayerStruggle()`, and the device-name/fallback resolution eventually creates a matching door, so softlock risk is low. The main fairness issue is privacy/permission ambiguity: modern iOS may return generic names, and the code falls back to `PLAYER` after five seconds. That fallback is practical, but it can fire before the player understands that the real device name was supposed to matter. One concrete improvement: delay or gate the fallback behind an explicit "name unavailable" clue so the first read still feels like a device-feature moment.
