Level 26 — Change the device's system language to unscramble the signs and swap the dead-end route for the real one to the exit.
VERDICT: polish (3/5)
STRENGTHS: Genuinely novel fourth-wall hook with a tasteful, non-spoiling t=0 panel ("not in a tongue this device still remembers") plus tri-lingual revert steps. The phone column is tight and readable, and the locale latch/baseline-relative gate logic is carefully thought through.
CONCERNS: On iPad the correct staircase is alpha=0/non-solid until solved, so the t=0 screenshot shows a huge dead void through the vertical center and an isolated exit plateau top-right — the "vertical fill" is invisible and the screen reads as broken. The leap from "scrambled signs" to "open iOS Settings and change the whole phone language" is large and unguided on first encounter.
CLUE/HINT: t=0 panel hints at language without naming the Settings trick, so it doesn't spoil it; hintText() gives the exact Settings path. But escalation is wired only to deaths (notePlayerStruggle on handleDeath), and a stuck player stands safely on the rest pad without dying, so the most likely stuck state never triggers help.
SUGGESTION: Give the hidden iPad staircase a faint low-alpha (still non-solid) ghost outline so the band reads as intentional and a route is visibly waiting to be revealed, and add a time-on-level trigger to notePlayerStruggle so a stuck-but-alive player still gets escalating hints.
