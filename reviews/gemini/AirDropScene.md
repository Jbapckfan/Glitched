Level 28 — Share to Decode
The terminal's scrambled text and share-to-decode mechanic cleverly utilize the device's share sheet to create an engaging cryptographic puzzle. The layout places the exit body out of reach at a 135pt vertical obstruction, ensuring the player must interact with the keypad to progress. The "TAKE ME SOMEWHERE I CAN [READ]" clue sets the stage effectively, allowing the player to discover the share sheet on their own before receiving explicit help. A notable code bug exists in createInGameKeyboard, which generates keypad buttons and assigns them a name like keyBtn_X_1, but fails to sanitize the randomized characters; if the target character happens to be an underscore, the components(separatedBy: "_") logic will extract the wrong substring. A concrete improvement is to store the target character in the node's userData dictionary instead of parsing it out of the node's name string to avoid delimiter collisions.
