Level 26 — Language / Locale
The scrambled unicode signs provide a clear, intriguing mystery that naturally prompts the player to change their device language. The zigzag switchback restricts lateral jumps and limits tier ascents to <=85pt, ensuring the hidden platforms are perfectly reachable once revealed. The visual "SOME MOMENTS REFUSE TO MOVE" tease works perfectly, and the multi-language settings prompt correctly addresses the eventual hurdle. A code bug occurs in onLocaleChanged: it uses an in-memory baselineLanguage to prevent auto-solving on launch, but if the device's baseline language is changed externally while the app is fully killed, the next cold boot will establish the new language as the baseline, preventing the puzzle from solving if the player switches back. A concrete improvement is to store the initial baseline language persistently in UserDefaults so the game always knows what language the puzzle was originally presented in.
